Enunciado de la pregunta

Se posee una barra de 1 m de longitud, la cual se encuentra inicialmente a una temperatura de 850°C. Considere que la ecuación diferencial que modela la difusión de la temperatura para el hierro es:



 a) Plantee una ecuación de discretización de la ecuación a derivadas parciales y obtenga un valor de Δx adecuado, considerando Δt=30 s. Indique el método utilizado, el valor de r y el valor de Δx. NO es necesario que transcriba el desarrollo para obtener dichos valores.

  b) La temperatura del borde izquierdo se reduce repentinamente y se fija en -25°C en t= 0 s, pero la temperatura del extremo derecho se mantiene en 850 °C.  Determinar el tiempo para el cual el nodo que se encuentra a 0.35 m de la superficie de -25°C adquiere una temperatura menor a 500°C. Indicar el método elegido.

 c) La temperatura del borde izquierdo se reduce siguiendo la ecuación:  Tiz=850./exp(t/100.)     (t: tiempo en segundos). La temperatura del extremo derecho se mantiene en 850°C. Determinar el tiempo para el cual el nodo que se encuentra a 0.35 m de la superficie izquierda adquiere una temperatura menor a 500°C. Indicar el método elegido.



A)

Utilizo el Metodo explicito, este metodo depende de los valores anteriores  y es un metodo  mas simple, para poder discretizar la ecuación diferencial se utiliza el método de  diferencias finitas (reemplazo las derivadas parciales por aproximaciones finitas)

La letra i indica la variable temporal (t) y la letra j indica la variable espacial (x)

DU/Dt= T (i, j+1) - T(i,j) /Dt

D^2U/dx^2 = T(i-1,j) - 2T(i,j) + T (i+1,j) / dx^2

Reemplazo en la ecuacion de derivadas parciales y sabiendo que r es igual a:

r=k.dt/c.p.dx^2

nos queda:

 U(i,j)=r*(U(i-1,j+1)+U(i-1,j-1))+(1.-2.*r)*(U(i-1,j))

notar que si r toma un valor de 0.5 la ecuación se simplifica quedando de la siguiente manera

U(i,j)=0.5*(U(i-1,j+1)+U(i-1,j-1))

para que r tome un valor de 0.5 en este caso, el dx tendra que tomar el valor de 0.036m



B) Para este caso use el metodo explicito tomando un dt=30s y un dx=0.05m dando un r=0.2734, y como r<0.5 no hay problemas de inestabilidad con este metodo.

 Con estas condiciones planteadas el tiempo que tarda el nodo a 0.35m en llegar a una temperatura menor que 500ºC  es de t=3780s y la temperatura fue de T=499.9ºC


C) para las condiciones planteadas en este caso tome los mismo  valores de dx=0.05, r=0.2734 y dt=30s, sigo usando el metodo explicito ya que no hay problemas de inestabilidad

con estas nuevas condiciones de contorno el nodo a 0.35m de la superficie izquierda llega a una temperatura menor a 500ºC en un tiempo  t=4110s y la temperatura fue T=499.12

Comentarios

Comentario:

a) El análisis es correcto. 

b y c) correctos.
