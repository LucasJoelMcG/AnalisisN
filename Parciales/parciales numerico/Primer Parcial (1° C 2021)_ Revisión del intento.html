Enunciado de la pregunta

Escriba un programa que permita calcular n valores de la función f(x)= x2.e-x desde un valor inicial X0 hasta un valor final Xf. Los valores de x y de f(x) deben ser almacenados en un archivo. El programa debe graficar la función f(x) por medio de gnuplot.

MODULE parcial

IMPLICIT NONE

CONTAINS

FUNCTION ejparcial(x)

real (8), parameter :: e=2.718281828

real (8) x,ejparcial

ejparcial=(x*(2))*e*(-x)

end function

END MODULE


PROGRAM principal_parcial

USE parcial

implicit none

real (8) x,y,xi,xf,avance

integer n

n=10     !cantidad de valores a calcular

xi=0   !valor inicial

xf=5    !valor final

x=xi

avance=(xf-xi)/(n-1.)

Open (unit=2, file='ejparcial.txt')

Do while (x<=xf)

 y=ejparcial(x)

 WRITE(*,'(2F15.6)', advance='No')x,y

 WRITE(2,'(2F15.6)', advance='No')x,y

 WRITE(,)' '

 WRITE(2,*)' '

 x=x+avance

END DO

Close (2)

call system ('gnuplot -persist grafico.p')



END PROGRAM

a.

Es discontínua y equiespaciada.

No es posible representar en forma exacta el valor  2(-0.5)

Es posible representar en un REAL(4) cualquier valor positivo siempre que se encuentre entre 1.1754944E-38 y 3.4028235E38
