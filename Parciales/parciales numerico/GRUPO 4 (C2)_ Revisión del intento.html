Enunciado de la pregunta

Se desea resolver la siguiente ecuación diferencial:        y’’+ 3.y - y’.cos(y) = 1

Las condiciones iniciales son: x0 = -1 ; y0 = 0 ; y’0 = 1

 1.  Escriba el código FORTRAN que describa el modelo planteado.

 2.  Calcule por el método de Euler simple, con h=0.1, el valor de y para x=4.

 3.  Recalcule el inciso anterior utilizando el método RK4 con h=0.1. Calcule la diferencia absoluta y relativa entre los valores de y para x=4 hallados mediante Euler simple y RK4. Justifique brevemente la diferencia encontrada (evite superar las 50 palabras, aprox.).

 4.  Utilizando el método de Euler simple calcule los valores de y para x=4. En este caso, utilice un ajuste de paso h que permita asegurar un error menor a 10-4. Calcule la diferencia absoluta y relativa entre los valores de y para x=4 hallados mediante Euler simple con y sin ajuste de paso h. Indique el menor valor de h obtenido en el intervalo establecido. Justifique brevemente la diferencia (evite superar las 50 palabras, aprox.).



1-planteando las condiciones necesarias la función que modela la ecuación planteada es


integer, PARAMETER:: cant_ec=2

  real(8), DIMENSION(0:cant_ec)::v_inicial

  v_inicial(0)=-1

  v_inicial(1)=0

  v_inicial(2)=1

  

    FUNCTION v_prima(v)

  REAL(8), DIMENSION(0:cant_ec) :: v, v_prima

  v_prima(0) = 1.0

  v_prima(1) =v(2)

  v_prima(2) = 1-(3*v(1))+(v(2)*cos(v(1)))

  END FUNCTION v_prima

3- el valor de y  usando RK4 con x=4 y h=0.1 es  Y= 3.03284097259

el error absoluto lo calculo como Yeuler-Yrk4  y es igual ERRORabs=1.6653

el error relativo lo calculo como ERRORabs/yrk4=  0.5496

al mantener el mismo h y usando distintos metodos (rk4  tiene menor error global que euler simple) podemos decir que el valor calculado con rk4 es mas preciso que euler y por eso la diferencia


4- el valor de y para Euler simple modificando el paso es  Y= 3.0925837637

el error absoluto lo calculo Y(h.0.1)-Y(h.modificado)=1.6055

el error relativo lo calculo como error abs/y(h.modificado)=0.5191

en el intervalo de 0 a 4 el h mas pequeño fue de 0.00156 con un valor inicla de h=0.1

al modificar el h (reducirlo ) hago que el error sea menor como el error global en euler simple es proporcional al valor de h al ir reduciendo el valor  el error tambien se reduce,  esto me permite  llegar a un valor aproximado de y   con euler simple  muy parecido al de rk4
